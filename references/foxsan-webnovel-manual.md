## 前言（摘录）

三年多写作经历，经历了三个阶段：
1. 初生牛犊期：一腔热血，从玄幻起步，第一部 80 万字烂尾但拿到全勤奖。
2. 扫榜突飞期：接触飞卢，疯狂扫榜写开头，开始真正入门，掌握市场与平台运作。
3. 反思转向期：转向中短篇与知乎盐选，《惊石》等作品进入云腾计划。

## 1. 网文基础知识

### 1.1 什么是网文
以互联网为展示与传播平台的文学作品；风格自由、题材宽泛、阅读简单、受众基数庞大。

### 1.2 类型
玄幻 / 奇幻 / 武侠 / 仙侠 / 都市 / 耽美 / 言情 / 历史 / 军事 / 游戏 / 体育 / 科幻 / 灵异 / 同人 等。

### 1.3 流派
神豪文、娱乐文、系统文、种田文、签到文、打卡文、四合院、明星文、挖矿文、年代重生、轻小说等。

### 1.4 发布平台
起点、晋江、番茄、七猫、飞卢、掌阅、云起、潇湘、磨铁、纵横、创世、红袖、火星、17K、阿里、QQ 阅读、黑岩、刺猬猫、咪咕、爱奇艺、铁血、红薯、塔读、九库、书旗、米读……

## 2. 如何开始写作

### 2.1 了解网文风向
- 手机里必须有：起点、晋江、番茄、七猫、飞卢；灵异加火星，战争加铁血。
- 不要只看一个平台；对比各平台榜单类型分布。

### 2.2 确定自己的方向
1. 不写极端冷门类型。
2. 按自己爱好选（写作是苦修，需要爱好支撑）。
3. 按经历/特长选（学生 → 校园/游戏；上班族 → 重生/职场；白领 → 总裁；军人 → 军旅）。
4. 按时间与打字速度选（飞卢开头 12 万字要一周，日码 1.5–2 万字）。
5. 按写作特点选（稳重型 ≠ 飞卢风）。

### 2.3 选择合适平台
- 先给自己定位，再找平台。
- 对比目标平台榜单类型占比；想写的类型是否在榜、占比多少。
- 例：想写灵异，起点没有 → 去火星。
- 飞卢：脑洞大、码字快、开头节奏快；起点：风格成熟、节奏可慢。
- 不要迷信"龙空"或盲目崇拜超级平台；新人最需要读者认可，不要只看"大神标准"。

## 3. 如何快速学习入门

### 3.1 混作者圈子
- 多听多看，不要一上来问"帮我看看写得怎么样"。
- 别装逼，拿作品说话。
- 别随便问别人作品；网文圈大神就那几个，多数人是平庸的。
- 作者发的牢骚（骂网站/编辑/作者）听听即可，不要人云亦云。

### 3.2 扫榜

#### 3.2.1 错误方法
只盯《遮天》《斗破苍穹》这种大神作品——大神有百万粉丝垫底，技巧已成熟，节奏慢也有人等，新人学不来。

#### 3.2.2 为什么不看大神
- 大神随便写都有人看；你是新人，没有这个特权。
- 大神节奏慢、铺垫长；新人没资本让读者等。
- 记住：你在最底层，就要学同样底层但刚刚签约的那批人写的是什么。

#### 3.2.3 正确方法
- **目的性**：扫自己感兴趣、想写的类型。
- **选择性**：扫一个网站的新书榜、畅销榜、飞卢天榜；重点扫"榜单里只有当前这一本书的作者"（即新签约作者）。
- **技巧性**：每本看前 10 章（重点前 5 章）：
  - 第 1 章：怎么开篇
  - 第 3 章：怎么铺垫
  - 第 5 章：怎么写完第一个小高潮，再往后铺垫
  - 重点看如何设计悬念、拉期待、写高潮；不要过度关注文笔。
  - 建立素材库 / 前十章表格，记录每章讲了什么。

### 3.3 如何拆书
- 狠拆：拆到一地碎渣，不能只看爽。
- 带目的地拆：人设、切入点、脉络、期待感设计等。
- 前 5 章要按段拆：前 200 字 / 前 500 字 / 前 1000 字各写了什么。
- 好梗要前后拆 2–3 章：铺垫 → 使用 → 升华 → 承接，整套记下来。
- 对话和打斗也要前后看：对话的作用、打斗的铺垫与爽点。

### 3.4 素材积累
- 开头描写、人设、等级/系统/地图设计、打斗场面、人物描写、好梗 → 全部进素材表。
- 工具要能随时记录（坐地铁、公交时）。
- 另建"灵感库"：不管有没有用都记，类似郭德纲的便签习惯。

## 4. 如何快速写出作品

### 4.1 梳理主线
1. 对标当下网文风向，不落伍、不俗套。
2. 把主线的维度拉开，保证后期能撑起一个庞大世界。
3. 主线分割成层层递进的方程式，找到切入点，明确主要矛盾。
4. 主线的设计 = 矛盾的设计。

### 4.2 人设
- 开篇只设计主角 + 1–2 个长期配角。
- 主角必须有缺点（人无完人，便于代入）；爽点 = 主角不断克服弱点的过程。
- 配角为主角服务，要鲜明但不抢主角光环。
- 可写小传：生活背景、成长环境、性格特点、人物关系、前世今生。

### 4.3 起名（11 类）

| # | 类型 | 代表作 | 特点 / 注意 |
|---|------|--------|------------|
| 1 | 人名型 | 《张三丰异界游》《张辽新传》《悟空传》 | 名人才有效，同人多 |
| 2 | 职业型 | 《武神》《狂神》《药王》《极品家丁》 | 职业必须人所熟知 |
| 3 | 技能型 | 《诛仙》《斗破苍穹》 | 笼统，限制后续构思 |
| 4 | 年代型 | 《回到明朝当王爷》《重生1998》 | 自带期待，历史/年代重生常用 |
| 5 | 地点型 | 《斗罗大陆》 | 指向不明，除非蹭大神热度 |
| 6 | 主线型 | 《七界武神》《异界之极品奶爸》 | 主线明显但受众受限 |
| 7 | 分类型 | 《重生之…》《四合院之…》 | 指向明确，局限强 |
| 8 | 期待型 | 《史上第一妖》《鬼吹灯》《宋朝没事》 | 吸引强，起名难度高 |
| 9 | 历程型 | 《凡人修仙传》 | 点明主角发展路径 |
| 10 | 东方型 | 《莽荒纪》《佛本是道》 | 东方色彩浓，现今不太适合 |
| 11 | 反衬型 | 《平平无奇大师兄》《师兄太稳健了》 | 风险高，会过滤读者 |

建议：配合扫榜起名；优先带期待感；名字不要太多暗喻；指向要直接。

### 4.4 大纲
- 主线（=主要矛盾）
- 支线（推动剧情，服务主线）
- 人设（出身、教养、技能、性格、缺点）
- 背景（空间、时间、玄幻等级、系统功能）

### 4.5 简介要素
背景、主要人物、冲突、悬念；语言准确明了；给读者一个非读不可的理由。

> 示例（原文）：刚穿越到玄幻世界，就遭遇气运之子即将打脸的情况？顾长歌表示有点懵。不仅女主被她圣主老爹亲自送到自己身边，就连和气运之子许下三年之约的前未婚妻，都对自己无比仰慕？啧，开局就拉满了气运之子的仇恨值……

### 4.6 剧情设计（签约前 & 签约后）

签约前（前 10 万字内）：
- **4.6.1 切入点**：故事开始的背景与事件，必须即将对主角产生巨大影响。
  例：主角家族被灭沦为奴，切入点定在奴隶主身死、需要大量奴隶殉葬。
- **4.6.2 第一个梗**：由题材/名字决定。战神归来 → 战神身份；重生 → 前世记忆；系统文 → 新手大礼包。
- **4.6.3 剧情为第一个梗服务**：利用第一个梗的矛盾设计第一个小剧情 → 第一个爽点 → 快速进入大剧情 → 前五章期待感立起来。
- **4.6.4 代入感**：用画面感 + 同情/共鸣/可怜等情感把读者与主角绑定。例：战神归来 → 先写战神强大 → 再写妻女受苦 → 从丈夫/爸爸角度描写。

签约后：
- **4.6.5** 快速完成整体构思（世界/等级/系统/势力）。
- **4.6.6** 分剧情：以第 5 章铺垫为起点，后 10 章一个轮回（约 3 万字）。
- **4.6.7** 10–15 万字时铺垫超级大高潮，进入付费章节，订阅爆发点。

### 4.7 投稿
- 找准定位 → 看平台需要什么风格 → 双方合拍。
- 新人不必只盯起点、晋江：读者挑剔，大神太多，新人不易出头。
- 投稿前先内投（结识编辑；问收稿类型；拒稿可问原因；不行再投别家，避免被盗）。
- 编辑一般用 QQ 邮箱；最快当天、一般 3 个工作日；半个月无消息可放弃。

## 5. 如何提高写作水平

### 5.1 爽点

> 核心：**脸挣足，面给够**。"佛争一炷香，人活一口气"。

**生理向**（男频）：
- 别人比我强 → 我会惊天武技 / 获得惊天秘籍（打脸）
- 别人比我帅 → 我身边美女如云（异性吸引，要把握度）

**心理向**：
- 有钱有势有女人（虚荣心）
- 你没有我有、你不知我知（站在世界最高处；《庆余年》范闲作诗）
- 反败为胜、扮猪吃虎（弱势反击）
- 全球瞩目、武林至尊（成为世界焦点）
- 嘚瑟无罪（招人妒恨，读者爽）
- 惊天实力、挡都挡不住（让人敬畏又恐惧）
- 实力不外露（爷很牛逼，不告诉你）

**第一个爽点**：在作品名字里 + 在作品类型里。
- 例：《神豪：开局我有10个亿！》→ 如何得到 10 亿、一步登天、不劳而获。
- 例：架空历史穿越 → 现代人如何利用现代智慧碾压古人。

### 5.2 文笔

**三层次**：
1. **规范**（无病句 / 错字 ≤ 5/千字）
2. **准确**（无歧义，用词精准）
3. **有感染力**（画面感 + 共鸣）

**提高规范**：
- 用查错软件 + 大声朗读。
- 多用短句 → 但避免高重复率 → 穿插倒装句。

**提高准确**：
- 替换常用词（说 → 道 → 讲；喜欢 → 欢喜 → 喜爱；红 → 珊瑚红/砖红/石榴红/银红桃……）
- 描写分类各总结 5 个常用模板（外貌/性格/穿着/动作/景色），背熟后随意结合。

**提高感染力**：
- 画面感：像 VR 一样，以一个点为原点旋转/延伸，调动视觉、嗅觉、触觉。
- 共鸣：围绕真理、情感等易共鸣的点来描写。

原文示例（节选）：
- 打扮：珠围翠绕、楚楚衣衫、含娇倚榻、花枝招展……
- 人物：旧法兰绒外套经过浸湿烤干这两重水深火热的痛苦，疲软肥肿……
- 画面感：弄堂口漆黑一片、冷风腐朽味、后脖颈凉意、汗毛倒竖。
- 共鸣：车开到楼下、坐在车里抽完一根烟再上楼。

### 5.3 剧情设计（以矛盾驱动）

核心原则：
- 新手本末倒置：往现有剧情里硬加爽点；**正确顺序：矛盾 → 剧情 → 爽点**。
- 爽点是矛盾解决过程与结果自然产生的。
- 好梗 = 雷人经典的爽被沉淀下来。

**示例：《玄幻：重生成了万年老祖！》**

1. 从名字看，读者已形成基础期待：玄幻题材 / 重生切入 / 成了一方老祖。
2. 顺着读者想法找矛盾 → 第一个矛盾：了解世界 + 冒充老祖。
3. 小剧情 1：两名弟子前来查看，主角用现代智慧糊弄过去。
4. 小剧情 2（升级）：疼爱的徒孙女看出破绽，双重压力，再次用现代智慧秒杀。
5. 大剧情：魔宗来犯，需主角启动护山大阵 → 闯禁地 → 猜谜/算法破阵 → 众人崇信。
6. 超级剧情：魔祖亲至，宗门再危 → 付费章节收官矛盾。

**轮回结构**：身份 → 实力 → 宗派 → 大陆 → 换地图，矛盾越来越大，剧情一浪盖一浪。

### 5.4 节奏（八步事件法）

1. **开局事件**：讲背景 + 主角身份，强调危机，主角试图逃避。
2. **升级事件**：危机升级，主角不得不迎战。
3. **结果事件**：主角虚假/暂时解决危机。
4. **转折事件**：危机出现变数，更不可控。
5. **调整事件**：主角积极转变，调整目标或手段。
6. **爆发事件**：危机疯狂爆发，主角陷入两难。
7. **高潮事件**：主角全力解决，场面宏大、震撼。
8. **结局事件**：危机解除，安放读者情绪。

## 6. 十大误区

1. **开头堆设定**：老书虫看的是故事，不是设定。
2. **开头不见主角**：没主角就没故事；背景可在故事推开后慢慢交代。
3. **主角被动瞎逛**：缺主线与层层矛盾，主角没有目的地闲逛。
4. **为了爽而爽**：硬塞爽点；正确做法是爽点从矛盾中自然生发。
5. **文笔决定一切**：网文节奏快，读者看讲故事能力，语言顺畅 > 华丽辞藻。
6. **目空一切**：写自己作品后看不进别人作品；"三人行必有我师"。
7. **不会写大纲**：没大纲说明没通篇考虑，矛盾未深挖。
8. **苦写不思考**：看教程却不对照自作品反思，写十几个开头仍不进步。
9. **扫书看热闹**：看书只图乐子，不提炼开头/节奏/技巧。
10. **过度关注平台**：新人没有挑平台的权利；重点是作品本身。
